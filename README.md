# Sitio Web Trinkwasser

## 🧱 Publicación en GitHub
1. Crear repositorio `trinkwasser-web`
2. Subir:
   - index.html  
   - style.css  
   - /img  
   - /scripts  
   - README.md  

---

## 🚀 Deploy en Vercel
1. Ir a https://vercel.com/dashboard
2. Add New Project
3. Selecciona `trinkwasser-web`
4. Deploy

---

## 🌐 Conectar dominio Akky → Vercel

### En Akky agregar:

| Tipo | Nombre | Valor |
|------|---------|--------|
| A | @ | 76.76.21.21 |
| CNAME | www | cname.vercel-dns.com |

Luego en Vercel:  
Settings → Domains → agregar:  
`trinkwasser.com.mx`
